## 📊 Project Overview – Motor Vehicle Theft Analysis 🚓

### 📌 Project Description

This project explores a dataset on **motor vehicle theft** to uncover trends, patterns, and risk factors associated with stolen vehicles.

The analysis provides insights into:

* Theft frequency over time
* High-risk locations and regions
* Most stolen vehicle types
* Time-based patterns (daily, monthly, seasonal trends)
* Crime distribution and hotspots

The objective is to transform raw crime data into meaningful insights that can support **security improvements, law enforcement strategies, and policy decisions**.

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing using SQL
* Aggregation of theft incidents by region and category
* Time-series analysis (monthly/yearly trends)
* Identification of high-risk vehicle types
* Ranking locations with the highest theft rates
* Use of JOINs to combine multiple datasets
* Subqueries and window functions for deeper insights

## 🎯 Key Insights & Findings

* Certain regions have significantly higher vehicle theft rates
* Specific vehicle types are more frequently targeted
* Theft incidents show noticeable seasonal and monthly trends
* Peak theft periods occur at specific times of the day
* Data patterns can support predictive crime analysis
